# v0.10.0 — 2026-05-08

Updated Vox Populi to 5.2.5. Savegame compatible.

## Narrators & Recording
- Added the narrator workflow, including episode parsing, assembly sessions, stage prompts, and a narrator console.
- Added OBS recording support with segmented output, dynamic folders, render-event capture, and strategist-session integration.
- Added process-management helpers for launching Civ, coordinating recording mode, and shutting services down cleanly.

## Oracle
- Revised Oracle mode with replay and livestream-oriented flows.
- Added replay cache support and improved retrieval/output handling.
- Added experimental batch processing with multi-provider support, including OpenAI and Google/Vertex batch paths.

## Civ Launch & Mod Loading
- Added automatic mod-loading templates that avoid editing vanilla Civ files.
- Improved DLL hook handling between OBS and non-OBS modes.
- Added random-seed plumbing for launch/session setup.

## MCP Server & Game Context
- Added production-mode action support and richer strategic-resource context.
- Improved player visibility around tradeable resources.
- Expanded tactical AI influence documentation and strategy-action feedback.
- Linked player summaries more directly to telemetry.

## Models & Providers
- Updated provider compatibility for Vercel AI 6.x and Vertex flex tier.
- Added or refreshed model support for GLM 5/5.1, Gemma 4 VLLM, Qwen 3.6 small models, and Salt.

## Infrastructure
- Refactored shared console setup across agents.
- Improved configuration handling, retry behavior, and tool-schema logging.
- Updated DLL builds and supporting release metadata.
